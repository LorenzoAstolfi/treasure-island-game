This is a simple text-based adventure game written in Python.

## How to play
1. Make sure you have Python 3 installed on your computer.
2. Download the file `treasure_island.py`.
3. Open a terminal or command prompt.
4. Go to the folder where the file is saved.
5. Run the game using this command:

   python treasure_island.py

6. Follow the instructions and type your choices in the terminal.

## Requirements
- Python 3
- A computer and a terminal

Have fun!
